## What This Is

`crystalclear-skills` is a library of modular, versioned AI agent skills with built-in human-in-the-loop (HITL) governance. Each skill is a self-contained unit: a system prompt, an input/output schema, HITL routing rules, and examples — all packaged in a format that drops directly into Claude, n8n, or any LLM framework.

This is the public open-source face of the **CrystalClearHouse Skill OS** — the infrastructure layer that governs how autonomous agents act in production.
